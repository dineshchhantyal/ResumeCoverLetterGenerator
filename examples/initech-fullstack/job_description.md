# Full Stack Engineer

**Company:** Initech
**Location:** San Francisco, CA (Hybrid)
**Level:** Early Career (0 to 3 years)

## About Initech

Initech builds web tools that help small teams manage their work without the bloat of enterprise software. We are a small, product focused engineering team that ships fast and owns features end to end. Our engineers work across the stack, from the database up to the browser, and we care about reliable, well tested code that scales with our growing user base.

## What You Will Do

- Build and ship end to end web applications, from data models to user facing screens.
- Develop and maintain backend services in FastAPI or Node.js, exposing clean REST and GraphQL APIs.
- Create responsive frontends in React that are fast, accessible, and easy to maintain.
- Design and evolve PostgreSQL schemas, and optimize queries as our data grows.
- Containerize services with Docker and deploy them to AWS.
- Improve our CI/CD pipelines so we can ship safely and often.
- Collaborate across the team to scope features, review code, and raise the bar on quality.

## Requirements

- Hands on experience building full stack web applications.
- Proficiency with **React** on the frontend and **FastAPI** or **Node.js** on the backend.
- Solid working knowledge of **PostgreSQL** and relational data modeling.
- Experience designing and consuming **REST** and **GraphQL** APIs.
- Familiarity with **Docker** and deploying to **AWS**.
- Comfort with **CI/CD** workflows and automated testing.
- Strong fundamentals in a typed language such as TypeScript or Python.

## Nice to Have

- Experience with Redis or other caching layers.
- Exposure to infrastructure as code or container orchestration.
- Open source contributions or side projects shipped end to end.
- AWS certification.

We are an early career friendly team. If you have shipped real software across the stack and want to own features end to end, we want to hear from you.
