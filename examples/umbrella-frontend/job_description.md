# Frontend Engineer

**Company:** Umbrella Web
**Location:** San Francisco, CA (Hybrid)
**Level:** Early Career

## About Umbrella Web

Umbrella Web builds customer-facing web products used by teams to plan, track, and ship work. We care about fast, accessible, and polished user interfaces, and we ship in small increments backed by tests. Our frontend team owns the experience end to end, from component design to data integration, and works closely with design and backend partners.

## Responsibilities

- Build and maintain responsive user interfaces in React and Next.js with TypeScript.
- Translate design mockups into reusable, accessible components styled with Tailwind CSS.
- Manage application state with Redux and integrate with REST and GraphQL APIs.
- Write component and integration tests to keep the UI reliable as it grows.
- Collaborate with designers and backend engineers to ship features end to end.
- Profile and improve page load and rendering performance across devices.

## Requirements

- Strong working knowledge of React, Next.js, and TypeScript.
- Experience styling responsive UIs with Tailwind CSS.
- Familiarity with state management using Redux.
- Experience integrating frontends with REST and GraphQL APIs.
- Comfort writing component tests and using CI to keep the UI green.
- Solid fundamentals in JavaScript, HTML, and CSS.
- 0 to 2 years of professional experience or equivalent project work.

## Nice to have

- Experience with OAuth based authentication flows.
- Exposure to Node.js backends and PostgreSQL.
- Familiarity with Docker and CI/CD pipelines.
- Open-source contributions or a portfolio of shipped projects.
