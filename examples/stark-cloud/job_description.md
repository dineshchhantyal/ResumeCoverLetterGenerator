# Cloud Engineer (Early Career)

**Company:** Stark Industries
**Location:** San Francisco, CA (Hybrid)
**Level:** Early Career / New Grad

## About Stark Industries

Stark Industries builds the cloud platform that powers our internal product teams. Our infrastructure team keeps services fast, reliable, and cost efficient across a growing AWS footprint. We are looking for an early career Cloud Engineer who is excited to learn infrastructure as code, automate deployments, and help us scale responsibly.

## Responsibilities

- Operate and improve workloads on AWS, including Lambda, EC2, ECS, and S3.
- Build and maintain infrastructure as code with Terraform so environments are repeatable and reviewable.
- Improve CI/CD pipelines that build, test, and deploy services automatically.
- Help monitor reliability, respond to incidents, and reduce production issues over time.
- Track and optimize cloud spend, finding ways to cut cost without hurting performance.
- Support basic networking setup such as VPCs, security groups, and load balancing.

## Requirements

- Hands on experience with AWS services such as Lambda, EC2, ECS, and S3.
- Familiarity with infrastructure as code, ideally Terraform.
- Comfort building and improving CI/CD pipelines.
- Understanding of networking basics and a focus on reliability.
- Strong scripting ability in Python or a similar language.
- Comfort with Docker and containerized services.

## Nice to Have

- Experience optimizing cloud cost.
- AWS certification.
- Familiarity with Linux and Bash.
- Exposure to container orchestration.

We welcome new graduates and early career engineers. If you are eager to learn cloud infrastructure and care about reliable, cost efficient systems, we want to hear from you.
